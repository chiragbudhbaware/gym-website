# 🏋️ ApexFitness Gym Website

A modern and responsive **Gym website** built using **HTML, CSS, and JavaScript**.
This project is designed as a business website for gyms to showcase their services, trainers, membership plans, and contact information.

The website includes interactive features such as a **mobile navigation menu, smooth scrolling, scroll-to-top button, and contact form interaction**.
